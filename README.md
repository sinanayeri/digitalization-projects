# Digitalization Projects

This repository contains digitalization and data projects with a strong focus on **Power BI**, **data modeling**, and **business-oriented analytics**.

The goal is to demonstrate how structured data, clean models, and well-defined KPIs can support operational and process-driven decision-making.

---

## Focus Areas

- Power BI dashboards for operational reporting
- Data modeling (star schema, relational models)
- Data quality checks and validation
- DAX measures and KPI design
- Process and digitalization-oriented analytics

---

## Repository Structure

- `powerbi/`  
  Power BI dashboards, DAX logic, and model documentation

- `data/`  
  Sample raw and processed datasets used for reporting

- `docs/`  
  Business context, assumptions, and digitalization concepts

---

## Data & Privacy

All data used in this repository is **anonymized, simulated, or publicly available**.  
No real company or personal data is included.

---

## About

This repository reflects practical experience in digitalization, analytics, and data-driven process optimization, combining technical implementation with business understanding.
